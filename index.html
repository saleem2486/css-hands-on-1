What is CSS?

  1. CSS stands for Cascading Style Sheets.
  2. CSS describes how HTML elements are to be displayed on screen,paper,or in other media.
  3. CSS is not neccesary CSS saves a lot of work. can control the layout of multiple web pages all at once.
  4. External stylesheets are stored in CSS files.

Css can be added to HTML documents in 3 ways:

  • Inline by using the style attribute inside Html elements
  • Internal by using style element in the head section.
  • External by using a link to link to an external CSS files .However, in this tuteriol we use inline and internal styles, because this is easy to demonstrate .and easier for you to try it yourself.

When comparing CSS class vs ID, the difference is that CSS class applies a style to multiple elements.ID, on the other hand ,applies a style to one unique wlement.ID is also special imn the you can use a special URL to link directly to an element and It's used by JavaScript.

In CSS, selectors are used to target a specific element or range of elements on a web page.Once an element has targeted,a style or set can be applied to an element.

There is a wide range of selectors aveliable.TWo of most communly used are class and ID.Both are used to target elements to which a style should be applied.
